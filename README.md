## Project Description
NEXUS-Card is a high-fidelity, responsive, web-based digital workspace built with React, Vite, Tailwind CSS, and Supabase. The system serves as an interactive canvas allowing students to customize, preview, and submit a sci-fi cyberpunk-styled dynamic Identification Badge featuring responsive design, custom dynamic text scaling, real-time QR code generation, and custom layout styling attributes. Administrators are provided a centralized command dashboard to manage incoming requests, monitor real-time badge aspect renders, download high-resolution badge image assets, and manage entries with an interactive animated processing system.

</div>

---

## Project Objectives
* **Streamline Digital Badge Issuance:** To provide a centralized platform where students can seamlessly configure, design, and submit their identification metrics to a registry matrix.
* **Dynamic Real-Time Previewing:** To implement an interactive visual canvas where structural changes, dynamic color configurations, and layout properties update instantly in real time.
* **Automate Asset Generation:** To give administrators a high-resolution export mechanism capable of snapshotting multi-face layouts directly into sharp `.png` graphic packages.
* **NFC Chip Pre-Programming:** To dynamically render standard web URL payloads into distinct QR code matrices, providing an accessible gateway for administrators to program physical NFC hardware modules.

---

## Key Features
* **Futuristic Card Workspace:** An editable frontend control canvas equipped with independent color pickers for background, circuit line, and text color configurations.
* **Philippine Mobile Routing Normalization:** A smart entry form that automatically sanitizes 11-digit local phone entries, strips leading zeroes, and prepends a static international `+63` prefix on the live layout layer.
* **3D Perspective Aspect Toggling:** A CSS 3D matrix card wrapper allowing students to spin the rendering workspace 180 degrees to review branding aesthetics on both front and rear card faces.
* **Interactive Shredit System:** An admin-centric deletion modal featuring a custom SVG trash bin container with a stylized lid that triggers a looping animation while deletion queries process.

# The project demonstrates our understanding of:
- Full-Stack Serverless Architecture: Linking a React application directly to a real-time cloud data layer managed by Supabase.
- Container Query Responsive Frameworks: Utilizing innovative style engines to calculate text-scaling properties dynamically using reduction factor formulas.
- High-Fidelity UI/UX Engineering: Injecting global keyframe definitions directly into styling layers to manipulate graphic structures safely.
- Data Vector Management: Running targeted database queries to update profile states or filter local array streams on frontend layouts.
- Git Version Controls & Workflow: Organizing structured team source code repositories across separate specialized development environments.
